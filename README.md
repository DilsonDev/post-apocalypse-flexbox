# Post Apocalypse Flexbox

> Tricks list to help implement the flexbox features in a world where it is not supported.

## flex container
*Properties for the Parent*

- display
- flex-direction
- flex-wrap
- flex-flow
- justify-content
- align-items
- align-content

## flex items
*Properties for the Children*

- order
- flex-grow
- flex-shrink
- flex-basis
- flex
- align-self
